# :partying_face: 部署

::: details
什么是静态网站，静态网站有什么特点？<br>
静态网站是指由静态文件（如 HTML、CSS 和 JavaScript）组成的网站，这些文件在服务器端不需要进行动态处理或生成。每次用户请求页面时，服务器直接将这些文件发送给用户的浏览器进行渲染。
静态网站的特点
- **简单性**：无服务器端脚本或数据库，结构简答。
- **安全性**：无动态内容生成，安全性较高。
- **性能**：加载速度快，因为文件可以直接从服务器或CDN缓存中获取。
- **易于部署**：只需上传文件到服务器即可。
